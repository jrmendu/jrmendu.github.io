Ejercicio 1

  • a. Pedir un dato al usuario y comprobar si es mayor de edad
  • b. Comprobar qué día es y sacar un mensaje personalizado en cada caso
  • c. Pedir dos datos para realizar una división -> comprobar que el divisor (segundo número en a / b sería la b) no es cero a / 0 // Infinity -> no es número

a. Solución:

Método de resolución

  • Usamos 'prompt()' para pedir el dato -> número // edad
  • Bucle if -> comprobamos si es > 18 -> escribimos un mensaje
  • else -> escribimos otro mensaje

b. Solución:

Método de resolución

  • Creamos un bucle o varios con if, else if, else para comprobar los 7 días
  • Añadimos operador || para verificar minúsculas
  • Si hay una falta ortográfica hay un mensaje por defecto en 'else'

c. Solución:

Método de resolución

  • Pedimos los valores como en un ejercicio anterior con 'prompt'
  • Usamos un bucle if para comprobar si el divisor es cero
