document.getElementById - Caso de Uso:

Siempre que queramos encontrar un elemento identificado por id en el HTML. Si hubiese mas de uno con ese id, encuentra el primero. Devuelve el elemento HTML o 'null' si no lo encuentra.

document.write() - Caso de Uso:

Cuando queramos escribir una salida de datos, texto, etc. en la página. Importante: Si se usa 'document.write()' con la página cargada se sobreescribe el contenido. Devuelve el contenido que estamos pasando a document.write() ya sea texto o HTML.

window.alert() - Caso de Uso:

Sirve para mostrar notificaciones que se deben confirmar para continuar. Coloca el dato que le pasemos como texto en la caja de notificación y podemos usar 'alert()'.

console.log() - Caso de Uso:

Muestra avisos o mensajes en consola para los desarrolladores -> solo se ve en la consola. También existen 'console.warn()' para warning, 'console.error()' para errores y 'console.info()' es similar a 'console.log()'.

window.print() - Caso de Uso:

Abre las opciones de impresión en la página actual.
